# EazyGiteeNote

## 开发中

做一个简约实用，基于 Gitee 的学习笔记吧！

持续更新开发中...

[预览地址](https://note.n0ts.cn/)

## 使用教程

配置 /js/config.js 即可

## 如何更新？

除了 `js/config.js` 文件以外全部覆盖即可

## 笔记是怎样展示的？

全部 markdown 文件丢到仓库根目录即可！

参考：[note: 个人笔记 (gitee.com)](https://gitee.com/n0ts/note)
